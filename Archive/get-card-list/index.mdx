View a list of cards associated with a particular customer with our **get card list** API endpoint.

## The request View a customer's cards by sending a **GET** API request to the URI below. In the URI, replace **{customerId}** with the customer's ID; remember to use the prefix `cust\_` to your customer ID. ### Endpoint live <pre class="api-header-long-url"><p class="api-header"><span class="get-tag">GET</span>https://api2.checkout.com/v2/customers/{customerId}/cards</p></pre> ### Endpoint sandbox <pre class="api-header-long-url"><p class="api-header"><span class="get-tag">GET</span>https://sandbox.checkout.com/api2/v2/customers/{customerId}/cards</p></pre> ### Header and path parameters

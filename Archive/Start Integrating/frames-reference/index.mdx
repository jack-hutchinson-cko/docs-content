Frames dispatches events which can be handled for various purposes. The table below provides the full list of events that you can attach one or more handlers to.

Event constants on the JavaScript API are accessible via Frames.Events.

|
|

|

\`CARD_SUBMITTED\`

|

\`CARD_TOKENISED\`

|

Triggered after a card is tokenised. Returns an object containing the card token and card information such as payment methods, expiration date, the first 6 (bin) and last 4 digits of the credit card.

|
|

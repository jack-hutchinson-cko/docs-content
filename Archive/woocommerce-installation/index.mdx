Start accepting payments with Checkout.com, on your WooCommerce store. 

If you already have WooCommerce installed on your WordPress site, skip to Step 2. 1. Login to your \*\*wp-admin\*\* page. 2. In the sidebar, click \`Plugins > Add New\`. 3. Search for the WooCommerce plugin, install it, then activate it. 

1\. Clone Checkout.com's WooCommerce extension from \[Github\](https://github.com/CKOTech/checkout-woocommerce-plugin). 2. Unzip your file and copy and paste the \*\*checkoutapipayment\*\* folder to \`\[wordpress directory\]\\wp-content\\plugins\\\`. 3. Access the administration page of your WordPress site by navigating to your wp-admin page. 4. In the sidebar, select \`Plugins > Add New\`. 5. Search for the \*\*Checkout.com Payment Gateway (GW 3.0)\*\* plugin, install and activate it.

1\. \*\*Enter your keys\*\*. In the related fields, enter your \*\*Secret Key\*\* and your \*\*Public API Key\*\* — you can find these in \[The Hub\](https://hub.checkout.com/). 2. \*\*Select a transaction method\*\*. \*Decide which option is best for your business.\* - \*\*Authorize Only \*\* Transactions will only authorize and must be captured manually. - \*\*Authorize and Capture\*\* Transactions are authorized and captured automatically. 3. \*\*Is 3D?\*\* a. Select \*\*YES\*\* to enable 3D Secure charges. b. Select \*\*NO\*\* to disable them. \*To activate this feature, please contact your account manager.\* 4. \*\*Auto Capture Time\*\* Auto-capture time defines how long after an authorization that a capture will be made — the default time is 0.\* 5. \*\*Timeout:\*\* The default number of seconds to wait before the request times out. 6. \*\*Endpoint Mode URL\*\* Select LIVE to enable live mode 7. \*\*Enable Local Payment\*\* Select YES to enable Local Payments. Select NO to disable them. 8. \*\*Are you PCI compliant?\*\* Merchants without an API account must set Are you PCI compliant? to NO. 

In this case, like the majority of our merchants, you are probably not fully PCI compliant and should choose No to enable Checkout.js.

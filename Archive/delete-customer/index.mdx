Use this API endpoint to return a customer's information. 

\## The request Delete a customer's information by sending a \*\*DELETE\*\* API request to the URI below. In the URI, replace \*\*{customerId}\*\* with the customer's ID. Remember to use the prefix \`cust\_\` to your customer ID. ### Endpoint live <pre class="api-header-long-url"><p class="api-header"><span class="delete-tag">DELETE</span>https://api2.checkout.com/v2/customers/{customerId}</p></pre> ### Endpoint sandbox <pre class="api-header-long-url"><p class="api-header"><span class="delete-tag">DELETE</span>https://sandbox.checkout.com/api2/v2/customers/{customerId}</p></pre> ### Header and path parameters

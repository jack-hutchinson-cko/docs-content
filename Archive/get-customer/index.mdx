Use this API endpoint to return a customer's information.

## The request Update a customer's information by sending a **GET** API request to the URI below. In the URI, replace **{customerId}** with the customer's ID. Remember to use the prefix `cust\_` to your customer ID. If you don't have the customer's ID, you can use their registered email address instead. ### Endpoint live <pre class="api-header-long-url"><p class="api-header"><span class="get-tag">GET</span>https://api2.checkout.com/v2/customers/{customerId|email}</p></pre> ### Endpoint sandbox <pre class="api-header-long-url"><p class="api-header"><span class="get-tag">GET</span>https://sandbox.checkout.com/api2/v2/customers/{customerId|email}</p></pre> ### Header and path parameters

## Product payload fields

Add Row Add Col

|     |     |
| --- | --- |

|
`description` <span class="field-type">String</span> <span class="optional-def">Optional</span>

|

Description of the product. Maximum characters: 500.

|
|

`image` <span class="field-type">String</span> <span class="optional-def">Optional</span>

|

Link to the product image on the merchant's website. Maximum characters: 200.

|
|

`name` <span class="field-type">String</span> <span class="optional-def">Optional</span>

|

Name of product. Maximum characters: 200.

|
|

`price` <span class="field-type">Decimal</span> <span class="optional-def">Optional</span>

|

Product price per unit. Maximum digits: 6.

|
|

`quantity` <span class="field-type">Integer</span> <span class="optional-def">Optional</span>

|

Units of the product to be shipped. Maximum digits: 3.

|
|

`shippingCost` <span class="field-type">Decimal</span> <span class="optional-def">Optional</span>

|

Shipping costs. Maximum digits: 6.

|
|

`sku` <span class="field-type">String</span> <span class="optional-def">Optional</span>

|

*Stock Unit Identifier *or _unique product identifier_. Maximum characters: 100.

|
|

`trackingUrl` <span class="field-type">String</span> <span class="optional-def">Optional</span>

|

Third party tracking system URL. Maximum characters: 500.

This endpoint enables you to fetch and view relevant information and parameters related to a payment. To use this endpoint, make sure you have the `chargeId` of the payment in question.

## The request Use the details below to set up your request. ### Endpoint live <pre class="api-header-long-url"><p class="api-header"><span class="get-tag">GET</span>https://api2.checkout.com/v2/charges/{chargeId}</p></pre> ### Endpoint sandbox <pre class="api-header-long-url"><p class="api-header"><span class="get-tag">GET</span>https://sandbox.checkout.com/api2/v2/charges/{chargeId}</p></pre> ### Header and path parameters

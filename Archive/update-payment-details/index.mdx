For whatever reason, sometimes you need to add or change some information related to a payment. Whereas not all payment information is editable, a few helpful bits are. You can update details for the `description`, `trackId`, `metadata`, and `udf1` to `udf5`.

Use this endpoint to update and add information to an existing payment. Before you start, you need the original `charge_id`; this is used to locate the payment you are updating. ## The request Use the details below to set up your request. ### Endpoint live <pre class="api-header-long-url"><p class="api-header"><span class="put-tag">PUT</span>https://api2.checkout.com/v2/charges/{chargeId}</p></pre> ### Endpoint sandbox <pre class="api-header-long-url"><p class="api-header"><span class="put-tag">PUT</span>https://sandbox.checkout.com/api2/v2/charges/{chargeId}</p></pre> ### Header and path parameters

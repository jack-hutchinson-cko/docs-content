Delete a card associated with a particular customer with our **delete card** API endpoint.

## The request Delete a card by sending a **DELETE** API request to the URI below. In the URI, replace **{customerId}** with the customer's ID and {cardId} with the card ID you wish to delete; remember to use the prefix `cust\_` to your customer ID and `card\_` for the card ID. ### Endpoint live <pre class="api-header-long-url"><p class="api-header"><span class="delete-tag">DELETE</span>https://api2.checkout.com/v2/customers/{customerId}/cards/{cardId}</p></pre> ### Endpoint sandbox <pre class="api-header-long-url"><p class="api-header"><span class="delete-tag">DELETE</span>https://sandbox.checkout.com/api2/v2/customers/{customerId}/cards/{cardId}</p></pre> ### Header and path parameters

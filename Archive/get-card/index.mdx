View the card details associated with a customer with our get card API endpoint.

## The request View a customer's card by sending a **GET** API request to the URI below. In the URI, replace **{customerId}** with the customer's ID, and **{cardId}** with the ID of the card you wish to update. Remember to add the prefix `cust\_` to your customer ID and `card\_` to the card ID. ### Endpoint live <pre class="api-header-long-url"><p class="api-header"><span class="get-tag">GET</span>https://api2.checkout.com/v2/customers/{customerId}/cards/{cardId}</p></pre> ### Endpoint sandbox <pre class="api-header-long-url"><p class="api-header"><span class="get-tag">GET</span>https://sandbox.checkout.com/api2/v2/customers/{customerId}/cards/{cardId}</p></pre> ### Header and path parameters

Use this API endpoint to update the metadata or track ID of a payment token. 

\## The request Update a payment token by sending a \*\*PUT\*\* API request to the URI below. In the URL, replace {paymentToken} with the payment token's ID. ### Endpoint <pre class="api-header-long-url"><p class="api-header"><span class="put-tag">PUT</span>https://api2.checkout.com/v2/tokens/payment/pay\_tok\_{paymentToken}</p></pre> ### Header and patch parameters

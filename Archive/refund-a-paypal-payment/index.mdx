Please refer to our \[Refund a payment\](doc:refund) for more information on how to refund a payment.

Refunds can be made using this API endpoint. Once processed, it is \*\*not possible\*\* to cancel a refund. ## The request Use the details below to set up your request. ### Endpoint live <pre class="api-header-long-url"><p class="api-header"><span class="post-tag">POST</span>https://api2.checkout.com/v2/charges/{chargeId}/refund</p></pre> ### Endpoint sandbox <pre class="api-header-long-url"><p class="api-header"><span class="post-tag">POST</span>https://sandbox.checkout.com/api2/v2/charges/{chargeId}/refund</p></pre> ### Header and path parameters

Use this API request to view all events and their type. ## The request Requests are created using a GET request. Use the details below to set up your request. ### Endpoint live <pre class="api-header-long-url"><p class="api-header"><span class="get-tag">GET</span>https://api.checkout.com/event-types </p></pre> ### Endpoint sandbox <pre class="api-header-long-url"><p class="api-header"><span class="get-tag">GET</span>https://api.sandbox.checkout.com/event-types </p></pre> ### Header parameters

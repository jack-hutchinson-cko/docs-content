Use this API endpoint to updates a new customer's information.

## The request Update a customer's information by sending a **PUT** API request to the URI below. In the URI, replace **{customerId}** with the customer's ID. Remember to use the prefix `cust\_` to your customer ID. ### Endpoint live <pre class="api-header-long-url"><p class="api-header"><span class="put-tag">PUT</span>https://api2.checkout.com/v2/customers/{customerId}</p></pre> ### Endpoint sandbox <pre class="api-header-long-url"><p class="api-header"><span class="put-tag">PUT</span>https://sandbox.checkout.com/api2/v2/customers/{customerId}</p></pre> ### Header parameters

|
|

`phone` <span class="field-type">Object</span> <span class="optional-def">Optional</span>

|

Phone number of the customer containing: _ `countryCode`: Valid country code for the phone number (e.g., 44 for the United Kingdom). _ `number`: Phone number. Permitted characters include: numbers, +, (,) ,/ and ' '. Must be between 6 and 25 characters. Example: "phone": {"countryCode": "44", "number": "12345678"} <span class="table-info">If countryCode is included, the number cannot be NULL.</span>

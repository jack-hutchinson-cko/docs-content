---
title: Magento 2: I’ve installed the plugin, but I get this error in the apache logs: “Class Checkout\Library\HttpHandler does not exist”. Why?
account: FAQ 
---


The plugin depends on our PHP software development kit (SDK). Make sure you [use composer to install the plugin](https://docs.checkout.com/quickstart/integrate/e-commerce-platforms/magento-2#Magento2-UsingComposercomposer), so it can also install the SDK.

Go to [GitHub](https://github.com/checkout/checkout-prestashop1.7-plugin/blob/drag-drop/checkoutcom.zip) and click **Download** to download ourÂ plugin. Log in to your PrestaShop admin dashboard and go to _Modules_ > _Module Manager_. Click **Upload a module** and drop the downloaded .zip file into the window that pops up. This will install the plugin.

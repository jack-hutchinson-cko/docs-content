The table below describes the possible response codes you can receive when sending an API request.

| Code | Description |
| ---- | ---- |
| `200` | OK |
| `201` | Created |
| `202` | Accepted |
| `401` | Unauthorized |
| `403` | Not allowed |
| `404` | Not found |
| `422` | Invalid data was sent |
| `429` | Too many requests or duplicate request detected |
| `502` | Bad gateway |

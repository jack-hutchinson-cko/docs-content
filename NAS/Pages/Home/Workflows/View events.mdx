Get a list of all event types, view details about a specific event, or get a list of all events related to a specific subject.

Get event types
---------------

Get a list of sources and their events to help you build new workflows.

### The GET request

Use the details below to set up your request.

#### Endpoints

*   [**Live**](#tabs-46e07525-c704-4be6-9ef5-64d75233be8c-1)
*   [**Sandbox**](#tabs-46e07525-c704-4be6-9ef5-64d75233be8c-2)

GET`https://api.checkout.com/workflows/event-types`

GET`https://api.sandbox.checkout.com/workflows/event-types`

#### Header parameters

Header

Value

Header

Value

`Authorization`

REQUIRED

`OAuth Bearer access token`

`Content-Type`

REQUIRED

`application/json`

### The GET response

Below is an example of a successful response.

    [
      {
        "id": "gateway",
        "display_name": "Gateway",
        "description": "Events from the Checkout.com payment gateway",
        "events": [
          {
            "id": "payment_approved",
            "display_name": "Payment Authorized",
            "description": "Occurs when a payment is successfully authorized by the Checkout.com payment gateway"
          }
        ]
      }
    ]

  

Other possible responses: 

*   401 - Unauthorized
*   422 - Invalid data was sent
*   500 - Internal error

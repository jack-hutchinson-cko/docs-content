After you've sent your payout request and got a successful response, you will receive a webhook notification that will tell you the status of your payout and include a response code and summary:

*   If your payout is [successful](#Payoutresponsecodes-paid), you will receive a `payment_paid` webhook, with a `10000` response code and the `Approved` summary.
*   If your payout is [declined](#Payoutresponsecodes-declined), you will receive a `payment_declined` webhook with a `50001-50399` response code and a summary explaining the reason for the decline.
*   If your payout is [returned](#Payoutresponsecodes-returned), you will receive a `payment_returned` webhook with `50401-50599` response code and a summary explaining the reason for the return.

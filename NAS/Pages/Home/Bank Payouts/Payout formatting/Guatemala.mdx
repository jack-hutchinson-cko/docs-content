Basic information
-----------------

  

Country code

GT

Supported currency

GTQ

Scheme

Local:

*   **Minimum amount (USD):** 10
*   **Maximum amount (USD):**
    *   **Banco Industrial**: 10,000 (per transaction, and for a single user per day or month)
    *   **Banrural**: 5,000 (per transaction, and for a single user per month)
*   **Cut-off time:** n/a
*   **Settlement:** T+0 same day

Payment information
-------------------

 Individual

### Beneficiary information

Beneficiary type

The type of account holder. Set this to `individual`.

`account_holder.type`

OPTIONAL

ENUM

Beneficiary name

The account holder's first and last name.

`account_holder.first_name`

REQUIRED

STRING

`account_holder.last_name`

REQUIRED

STRING

Beneficiary address

The account holder's address.

  

  

  

  

  

  

  

  

`account_holder.billing_address.address_line1`

REQUIRED

STRING

`account_holder.billing_address.address_line2`

OPTIONAL

STRING

`account_holder.billing_address.city`

REQUIRED

STRING

`account_holder.billing_address.zip`

OPTIONAL

STRING

`account_holder.billing_address.country`

REQUIRED

ENUM, 2 CHARACTERS

This must be a [two-letter ISO code](/display/DOCS/Country+codes).

Identification  
  

The account holder's ID.  
  

`account_holder.identification.type`

REQUIRED

ENUM

Can be one of `passport`, `driving_licence`, or `national_id`.

`account_holder.identification.number`

REQUIRED

STRING

If the identification type is `national_id` and the issuing country is `GT`, the number must be 13 digits long.

`account_holder.identification.issuing_country`

REQUIRED

ENUM, 2 CHARACTERS

This must be a [two-letter ISO code](/display/DOCS/Country+codes).

### Bank details

  

Account number

The account holder's account number.

`account_number`

REQUIRED

STRING, ^\[0-9\]{16,16}$

SWIFT BIC

The account holder's SWIFT BIC.

`swift_bic`

REQUIRED

STRING, ^\[A-Z\]{6}\[A-Z0-9\]{2}(\[A-Z0-9\]{3})?$
